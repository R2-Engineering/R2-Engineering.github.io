Microsoft 365 & Cloud Migration

  • Migrated 200 users from M365 Commercial to GCC to meet federal compliance requirements

  • Migrated 20TB of file storage from outdated on-prem servers to SharePoint
  • Replaced difficult-to-manage Datto Workplace solution with SharePoint
  • Created custom PowerShell scripts to automate migration tasks and validate data completeness

Security & Device Management

  • Deployed Microsoft Intune across 400+ devices for centralized management
  • Implemented Microsoft Defender for all users and devices
  • Configured and deployed BitLocker encryption on all Windows devices

  • Implemented Microsoft Purview for data governance and compliance monitoring
  • Set up Druva FedRAMP backup for server workloads and M365

  • Created comprehensive NIST 800-171 documentation including policies, procedures, network diagrams, and data flow diagrams

Infrastructure & Audit Preparation

  • Audited and upgraded Meraki network infrastructure for FIPS compliance
  • Replaced outdated file servers with Hyper-V host for archive and engineering workloads
  • Configured Azure Arc for on-prem server monitoring and management
  • Completed vulnerability scans and remediation across all devices
  • Conducted software inventory audit and removed unapproved applications using custom PowerShell automation
  • Established meeting structure and cadence for all required compliance reviews
